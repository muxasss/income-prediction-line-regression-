Income Prediction Line Regression Project
Описание проекта
Данный проект представляет собой веб-приложение для прогнозирования дохода на основе введённых пользователем данных. Прогноз осуществляется с использованием модели машинного обучения — множественной линейной регрессии.
Цель проекта
Целью проекта является разработка простой системы, позволяющей оценить предполагаемый доход человека на основе ключевых параметров, таких как возраст и опыт работы.
Проект демонстрирует интеграцию машинного обучения и веб-разработки в одном приложении.

Установка и запуск
1.⁠ ⁠Клонировать репозиторий:
git clone https://github.com/your-username/income-prediction-project.git
cd income-prediction-project
2.⁠ ⁠Создать виртуальное окружение:
python -m venv .venv
source .venv/bin/activate
3.⁠ ⁠Установить зависимости:
pip install flask pandas numpy scikit-learn matplotlib
4.⁠ ⁠Обучить модель (если model.pkl отсутствует):
python train.py
5.⁠ ⁠Запустить приложение:
python app.py
6.⁠ ⁠Открыть в браузере:
http://127.0.0.1:5000
Использование
Пользователь вводит следующие параметры:
•⁠ ⁠Возраст (Age)
•⁠ ⁠Опыт работы в годах (Experience)
После ввода данных система рассчитывает и отображает прогнозируемый доход в сомах (KGS).
Примечание
В процессе работы может появляться предупреждение, связанное с версией библиотеки scikit-learn. Это не влияет на корректность работы приложения.
