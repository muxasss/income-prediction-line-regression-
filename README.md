Income Prediction Line Regression Project
Описание проекта
Данный проект представляет собой веб-приложение для прогнозирования дохода человека на основе введённых пользователем данных. Прогноз осуществляется с использованием модели машинного обучения — множественной линейной регрессии.
Пользователь вводит возраст и опыт работы, после чего система рассчитывает предполагаемый уровень дохода.
Цель проекта
Целью проекта является разработка простой и наглядной системы прогнозирования дохода на основе ключевых параметров (возраст и опыт работы), а также демонстрация интеграции машинного обучения с веб-разработкой.
Проект показывает полный цикл работы:
обучение модели → сохранение → подключение к веб-приложению → получение предсказания через сайт.

Установка и запуск
Клонировать репозиторий:
git clone https://github.com/your-username/income-prediction-project.git
cd income-prediction-project
Создать виртуальное окружение:
python -m venv .venv
source .venv/bin/activate
Установить зависимости:
pip install flask pandas numpy scikit-learn matplotlib
Обучить модель (если model.pkl отсутствует):
python train.py
Запустить приложение:
python app.py
Открыть в браузере:
http://127.0.0.1:5000
Использование
Пользователь вводит следующие параметры:
Возраст (Age)
Опыт работы в годах (Experience)
После нажатия кнопки «Рассчитать» система отправляет данные на сервер, модель обрабатывает их и возвращает прогнозируемый доход, который отображается на странице без перезагрузки.
Также на сайте отображается пример датасета и график зависимости реальных и предсказанных значений.
Принцип работы
В файле train.py происходит обучение модели линейной регрессии на основе датасета.
Обученная модель сохраняется в файл model.pkl.
Файл app.py загружает модель и принимает данные от пользователя через POST-запрос.
Модель рассчитывает доход и возвращает результат в формате JSON.
JavaScript отображает полученный результат на странице.
